# BookStore 專題成品功能畫面呈現

## 功能描述
### 1. 會員註冊、登入及登出之功能
### 2. 購物車:可修改購買商品及訂購數量
### 3. 商城頁面包含、新品上架、熱門商品、限時特價......等
### 4. 具備後台可查看訂單、書籍商品管理、查詢會員資料等功能
